# Backup - Wysokie Tekstury Wojanek

Ten folder zawiera backup wysokich (tall) wersji tekstur napojów Wojanek przed przycięciem do kwadratu.

## Rozmiary:
- Oryginalne tall: 254x384 pikseli (proporcje butelki)
- Obecne kwadratowe: 256x256 pikseli

## Przeznaczenie:
Backup na wypadek gdyby trzeba było wrócić do proporcji tall dla tekstur napojów.

**UWAGA:** Te pliki NIE są pakowane do ZIPów dla serwera - są tylko w repozytorium GitHub.
