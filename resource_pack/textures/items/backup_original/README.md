# Backup - Oryginalne Tekstury

Ten folder zawiera oryginalne wysokiej rozdzielczości tekstury przed optymalizacją.

## Rozmiary:
- Oryginalne: 1024x1024 do 1092x1092 pikseli
- Zoptymalizowane (w folderze nadrzędnym): 256x256 pikseli

## Przeznaczenie:
Backup na wypadek gdyby w przyszłości były potrzebne tekstury wyższej rozdzielczości.

**UWAGA:** Te pliki NIE są pakowane do ZIPów dla serwera - są tylko w repozytorium GitHub.

## Oszczędności:
Optymalizacja tekstur zaoszczędziła **10.26 MB** (86% redukcja rozmiaru).
